## Brand Visual Guidelines

### Color Palette (from existing CSS theme)

| Name | Hex | Usage |
|------|-----|-------|
| Science Blue 900 | `#0a3d7c` | Primary headers, hero overlays |
| Science Blue 700 | `#1565c0` | Buttons, links |
| Science Blue 50 | `#e8f0fe` | Light backgrounds |
| White | `#ffffff` | Backgrounds, text on dark |
| Dark (gray-900) | `#111827` | Body text |

### Typography (from site)

- Headings: System sans-serif stack (clean, modern)
- Body: Same stack at regular weight

### Photography Style Guide

- **Tone:** Warm, authentic, intimate — not clinical or stock-looking
- **Lighting:** Natural light preferred. Soft, diffused. Avoid harsh flash.
- **Colors:** Earthy tones, blues, greens, warm neutrals
- **People:** Diverse, relaxed, genuine expressions. No forced poses.
- **Framing:** Leave breathing room. Don't crop too tight on groups.
- **Mood:** Contemplative, connected, grounded, hopeful
- **Avoid:** Overly staged setups, dark/gloomy moods, cluttered backgrounds, heavy filters

## Photo Release & Model Consent

A standardized photo release form is required for:
- All teachers (for their portraits on the website)
- All participants photographed during sessions
- Any person identifiable in community/group photos

➡️ **See full form:** [PHOTO_RELEASE.md](PHOTO_RELEASE.md)

### Distribution Procedure

1. **Before each intensive/session:**  
   - Print copies of the release form  
   - Include in registration materials  
   - Collect signed forms before any photography begins  

2. **For remote teachers:**  
   - Email PDF version  
   - Accept digital signature or scanned signed copy  
   - Store signed copies securely  

3. **Record keeping:**  
   - Maintain a spreadsheet tracking: Name, Date Signed, Event, File location  
   - Keep signed forms for minimum 5 years  
   - Store digitally in a shared drive with restricted access

## Production Checklist

### Phase 1 — Preparation (Before Photo Shoot)

- [ ] Hire photographer or designate staff photographer
- [ ] Brief photographer with this document + Brand Visual Guidelines
- [ ] Print Photo Release forms (30+ copies)
- [ ] Prepare shot list printout for the day
- [ ] Scout venue for best lighting / angles
- [ ] Prepare props: training manuals, journals, candles, cushions
- [ ] Confirm teachers willing to be photographed
- [ ] Schedule teacher portrait sessions (15 min each)

### Phase 2 — Shoot Day (During Intensive)

- [ ] **Collect signed release forms** from all participants at registration
- [ ] Shoot teacher portraits (B1–B12) — consistent background/lighting
- [ ] Shoot community/group photos (C1–C10) — candid + planned
- [ ] Shoot venue/environment (D1–D6) — before, during and after sessions
- [ ] Shoot behind-the-scenes for social media (S6, S11–S13)
- [ ] Shoot detail/symbolic shots (E4–E6) as time allows
- [ ] Capture short video clips for reels/stories (15–30 sec each)

### Phase 3 — Post-Production

- [ ] Select best shots (aim for 5–8 usable images per category)
- [ ] Edit/retouch for consistency (color grading, crop, contrast)
- [ ] Export for web: heroes at 1920×1080, portraits at 800×800, OG at 1200×630
- [ ] Export for print: 300 DPI versions for brochures/posters
- [ ] Rename files with descriptive names (e.g., `teacher-tina-lindhard.jpg`)
- [ ] Create OG default image (F1) — logo + tagline on branded background
- [ ] Generate favicon package (F2) from logo

### Phase 4 — Implementation (Website)

- [ ] Upload teacher portraits to `src/assets/images/people/teachers/`
- [ ] Update `coreTeachers.json` and `guestTeachers.json` image URLs to local paths
- [ ] Replace Unsplash hero images with originals in each HTML page
- [ ] Replace homepage community grid photos with new high-quality versions
- [ ] Add `og-default.jpg` to `src/assets/images/`
- [ ] Add favicon files to site root
- [ ] Add featured images to blog posts
- [ ] Test all pages for broken images
- [ ] Run Lighthouse audit for image optimization

### Phase 5 — Social Media Launch

- [ ] Create branded templates in Canva/Figma using Brand Visual Guidelines
- [ ] Prepare first 2 weeks of content using new photos
- [ ] Schedule posts using scheduling tool (Buffer, Later, etc.)
- [ ] Create Instagram Highlights: "About Us", "Techniques", "Teachers", "Events"
- [ ] Update profile photos/cover images on all platforms
