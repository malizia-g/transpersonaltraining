# Color Palette: Scientific Authority + Human Warmth

## Overview
This color palette combines **scientific credibility** (blue) with **human warmth** (yellow-tinted neutrals) and mystic depth (indigo & iris) to create a professional yet approachable brand identity that balances authority with soul.

---

## Primary Colors

### Scientific Blue (#1E40AF)
- **Role**: Primary brand color, main action buttons, core authority elements
- **Psychology**: Trust, stability, authority, intelligence, professionalism
- **Usage**: 
  - Primary buttons (white background for clean look)
  - Primary headings in sections
  - Link accents
  - Section backgrounds for authority

### Deep Indigo (#312E81)
- **Role**: Secondary heading color, depth and richness
- **Psychology**: Depth, wisdom, rich authority
- **Usage**: 
  - Card headings
  - Welcome section headings
  - Rich background overlays
  - Professional section titles

**Variations:**
- `science-blue-50` (#F0F6FF) - Backgrounds, very light
- `science-blue-100` (#E0ECFF) - Icon backgrounds
- `science-blue-300-600` - Accent ranges
- `indigo-deep-900` (#312E81) - Headings, depth

---

## Accent Colors

### Warm Yellow (#FCD34D)
- **Role**: Secondary accent, warmth indicator, badge highlights
- **Psychology**: Warmth, accessibility, optimism, humanity
- **Usage**:
  - Badge foregrounds (teachers core badges)
  - Icon accents
  - Hover states
  - Subtle warmth elements

**Variations:**
- `warm-yellow-100-600` - Full range

### Calm Teal (#2DD4BF)
- **Role**: Secondary accent for visual hierarchy, wellness/calm feeling
- **Psychology**: Calm, healing, growth, balance
- **Usage**:
  - Card top borders
  - Secondary icon colors
  - Supporting accent elements

## Iris/Mystical Section

## Design Principles Applied

### Trust (Blue) + Warmth (Warm Neutrals + Yellow)
- Blue conveys scientific credibility for therapists
- Warm tones show humanity and approachability
- Iris/violet adds spiritual depth

### Approachability with Authority
- Professional blue establishes credibility
- Warm backgrounds and accents show accessibility
- Iris/mystical elements reinforce transpersonal aspect

### Accessibility ✅
- White buttons on blue backgrounds: 9.6:1+ contrast
- All text meets WCAG AA standards
- Color-blind friendly (not relying solely on color)

## Implementation Summary

### Tailwind Color Classes
- `text-science-blue-*` / `bg-science-blue-*`
- `text-indigo-deep-*` / `bg-indigo-deep-*` (headings)
- `text-warm-yellow-*` / `bg-warm-yellow-*` (accents)
- `text-accent-teal-*` / `bg-accent-teal-*` (secondary)
- `text-neutral-warm-*` / `bg-neutral-warm-*` (text/backgrounds)
- `text-violet-*` / `bg-violet-*` (mystical sections)

### CSS Variables Available

- `--color-indigo-deep` (#312E81)
- `--color-cta` (#FCD34D)
- `--color-accent-teal` (#2DD4BF)
